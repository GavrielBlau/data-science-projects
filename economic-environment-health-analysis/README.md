# Economic, Environmental, and Health Data Analysis

This project analyzes relationships between economic indicators, environmental factors, and public health outcomes using multiple datasets. The analysis combines GDP data, green space sustainability metrics, and public health indicators to explore potential correlations between economic development, environmental conditions, and health outcomes.

The project demonstrates data cleaning, integration of multiple datasets, and exploratory analysis using R. Libraries such as dplyr, tidyr, ggplot2, and readr are used to prepare and visualize the data. The goal is to better understand how economic and environmental variables may influence health-related outcomes.
