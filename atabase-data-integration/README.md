# SQLite Data Integration Analysis

This project demonstrates how multiple datasets can be integrated and analyzed using a SQLite database and Python. The notebook connects to a SQLite database containing multiple tables, including weather data and crime reports, and loads the data into pandas DataFrames for analysis.

The project focuses on querying structured data from a database, inspecting table schemas, and combining information from multiple sources for further analysis. It demonstrates practical skills in database interaction, data extraction using SQL queries, and exploratory analysis using Python and pandas.
