# Traffic Signal Optimization Analysis

This project analyzes traffic signal control data to understand and predict reinforcement learning actions used in traffic signal optimization. The dataset includes variables such as queue lengths, average vehicle speed, signal phase, and phase duration.

Machine learning models including Random Forest and Gradient Boosting classifiers were used to predict the reinforcement learning action (`rl_action`) taken by the traffic control system. The project demonstrates data preprocessing, model evaluation, and feature importance analysis to understand which traffic conditions most strongly influence signal control decisions.
